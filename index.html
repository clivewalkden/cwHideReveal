This script enables the hide reveal ability that so many sites use.

Configuration Options

The script has 5 options:

  • speed
  • easing
  • changeText
  • showText
  • hideText
  • accordian

speed

The speed in which the animation completes.

This is the amount of time it takes the hidden div to be shown or hidden.

default: 300

easing

Easing type to use.

If you want easing in your animation include it here. If you include the jQuery UI easing plugin you will have more options available.

default: ''

changeText

Change the link text.

If you want to specify the text that the link element updates to on click then set this to true.

default: false

showText

The text shown to show content.

The text you want the button to display when the content is hidden.

default: 'Show

hideText

The text shown to hide content.

The text you want the button to display when the content is hidden.

default: 'Hide'

accordian

Hide other hidereveal content when 1 link is clicked.

If you want the script to act like an accordian and close other hidereveal content when another link is clicked then set this to true.

default: false

openClass

Class applied to open containers.

This is the class that is applied to containers once they have reached their open status.

default: false
